# Project 2: Exploratory Data Analysis (EDA)

## About the Project

This project was completed as part of the DecodeLabs Data Analytics Industrial Training Program. After cleaning the dataset in Project 1, the next step was to explore the data and understand its overall structure. The objective was to identify patterns, trends, distributions, and potential outliers using descriptive statistics and data visualization techniques.

## What I Did

During this project, I analyzed the cleaned dataset by performing the following tasks:

- Explored the overall structure of the dataset.
- Calculated descriptive statistics such as mean, median, mode, minimum, maximum, and standard deviation.
- Examined the distribution of numerical variables.
- Identified trends and relationships within the data.
- Detected potential outliers using statistical methods.
- Generated visualizations including histograms, boxplots, and a correlation heatmap.
- Summarized the key findings and observations from the analysis.

## Result

The exploratory analysis provided a better understanding of the dataset by highlighting important statistical measures, identifying outliers, and revealing trends across different variables. The findings serve as a strong foundation for future tasks such as dashboard development, predictive modeling, or business decision-making.
